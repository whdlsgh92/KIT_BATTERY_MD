# Battery Microdegree Landing Page (GitHub Pages)

이 폴더는 **빌드 없이** 바로 GitHub Pages로 배포 가능한 단일 페이지 템플릿입니다.

## 빠른 배포(추천)

### 1) GitHub 저장소 생성
- GitHub에서 새 Repository 생성
- 예: `battery-microdegree`

### 2) 파일 업로드
이 폴더의 파일을 저장소 루트에 업로드:
- `index.html`
- `styles.css`
- `script.js`
- `assets/` (이미지 폴더)

### 3) GitHub Pages 켜기
- 저장소 → **Settings** → **Pages**
- Source: **Deploy from a branch**
- Branch: `main` / Folder: `/ (root)` 선택 → Save
- 배포 URL이 생성됩니다.

## 수정 포인트(필수 3개)
1. `index.html`의 `YOUR_FORM_URL`을 실제 신청 폼 링크로 변경
2. `index.html`의 `YOUR_EMAIL@school.ac.kr` 이메일 변경
3. `index.html`의 전화번호 변경

## 이미지(선택)
- `assets/og.png`: 공유(카톡/페북) 미리보기 이미지 (권장 1200x630)
- `assets/favicon.svg`: 파비콘

## 커스터마이징
- 색상/폰트: `styles.css`의 `:root` 변수에서 변경
- 섹션 문구: `index.html`에서 바로 편집
